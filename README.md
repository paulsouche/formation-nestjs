# Formation NestJs

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## Install the dependencies (needs yarn v1.x.x)

```bash
$ yarn install
```

## Build the app

```bash
$ yarn build
```

## Run the app

```bash
# development
$ yarn start

# production mode
$ yarn build
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:coverage
```

## Lint

```bash
$ yarn lint
```

## Format

```bash
$ yarn format
```
